1.1 Instalación de R y RStudio
Antes de comenzar a escribir código, necesitamos instalar dos herramientas fundamentales:

R (El motor): El lenguaje de programación y entorno estadístico.

RStudio (El entorno/IDE): La interfaz gráfica cómoda y moderna que usaremos para programar.

📥 Paso 1: Descargar R (CRAN)
Para descargar R, debemos ir al sitio oficial del CRAN (Comprehensive R Archive Network).

Ingresa a la página oficial: https://cran.r-project.org/

En la parte superior, selecciona tu sistema operativo:

Download R for Windows (Si usas Windows)

Download R for (Mac) OS X (Si usas Mac)

Download R for Linux (Si usas Linux)

Si elegiste Windows, haz clic en el enlace que dice "base" (o "install R for the first time").

Finalmente, haz clic en el botón de descarga principal (por ejemplo: Download R 4.4.x for Windows). Esto bajará el instalador .exe.

💻 Paso 2: Descargar RStudio
Una vez que tengas el instalador de R, necesitas la interfaz para trabajar cómodamente.

Ingresa a la página oficial de Posit: https://posit.co/download/rstudio-desktop/

Desplázate hacia abajo hasta encontrar la sección "2: Install RStudio".

Haz clic en el botón azul "DOWNLOAD RSTUDIO DESKTOP FOR WINDOWS" (o el sistema operativo que detecte tu computadora).

🛠️ Paso 3: Instalación y Orden de Ejecución
⚠️ ¡IMPORTANTE! Siempre debes instalar primero R y después RStudio. De lo contrario, RStudio no sabrá dónde encontrar el motor para ejecutar tu código.

En Windows: Abre el archivo .exe de R, dale "Siguiente" a todo usando las opciones por defecto, y haz lo mismo con el instalador de RStudio.

En Mac: Abre los archivos .pkg o .dmg y arrastra RStudio a tu carpeta de Aplicaciones.

¡Listo! Una vez instalados ambos, solo tendrás que abrir RStudio para empezar a programar el resto de tu curso.
